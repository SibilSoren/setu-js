title: Quickstart
description: Get started with YantrJS in 3 minutes.
---

Get from zero to a production-ready API in minutes. YantrJS supports **Express**, **Hono**, and **Fastify**.

## 1. Create a New Project

Run the interactive wizard:

```bash
npx yantr-js create my-api
```

The wizard will guide you through:
1. **Runtime** — Node.js or Bun
2. **Framework** — Express, Hono, or Fastify
3. **Database** — PostgreSQL, MongoDB, or skip
4. **ORM** — Prisma, Drizzle, or Mongoose
5. **Components** — Auth, Logger, Security

Or use defaults for a quick start:

```bash
npx yantr-js create my-api --yes
cd my-api
```

## 2. (Alternative) Initialize in Existing Project

If you already have a Node.js project:

```bash
cd my-existing-project
npx yantr-js init
```

YantrJS will:
1. **Detect** your framework from `package.json`
2. **Create** `yantr.json` configuration
3. **Install** framework-specific base templates

## 3. Add Components

Add production-ready features as needed:

```bash
yantr add database  # Set up PostgreSQL/MongoDB with ORM
yantr add auth      # JWT authentication
yantr add logger    # Pino structured logging
yantr add security  # Rate limiting + Helmet headers
```

## 4. Generate Routes

Scaffold a new API resource:

```bash
yantr generate route users
```

Creates:
- `routes/users.routes.ts`
- `controllers/users.controller.ts`
- `services/users.service.ts`

## 5. Wire it up

### Express

```typescript
import express from 'express';
import { errorHandler } from './lib/yantr/error-handler';
import { httpLogger } from './lib/yantr/logger/http-logger';
import usersRoutes from './routes/users.routes';

const app = express();
app.use(express.json());
app.use(httpLogger);
app.use('/api/users', usersRoutes);
app.use(errorHandler); // Must be last

app.listen(3000);
```

### Hono

```typescript
import { Hono } from 'hono';
import { onError } from './lib/yantr/error-handler';
import usersRoutes from './routes/users.routes';

const app = new Hono();
app.onError(onError);
app.route('/api/users', usersRoutes);

export default app;
```

### Fastify

```typescript
import Fastify from 'fastify';
import { errorHandler } from './lib/yantr/error-handler';

const fastify = Fastify();
fastify.setErrorHandler(errorHandler);

fastify.listen({ port: 3000 });
```

That's it! You now have a production-ready foundation that you fully own.
